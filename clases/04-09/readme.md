# Clase 09 de abril de 2026

## Ejercicios de Lógica: Estructuras Repetitivas (Bucle for)

El objetivo de esta serie de ejercicios es dominar el uso de bucles anidados y condiciones lógicas para la generación de patrones visuales en consola.

Desarrollar un programa para cada uno de los siguientes patrones, solicitando al usuario (cuando sea necesario) el tamaño o la cantidad de líneas:

1.  **Cuadrado Sólido:** Generar una cuadrícula de $N \times N$.
2.  **Triángulo Rectángulo Invertido:** Dibujar un triángulo con la hipotenusa hacia la izquierda, reduciendo la cantidad de caracteres por fila.
3.  **Triángulo Rectángulo:** Dibujar un triángulo alineado a la izquierda que crezca fila a fila.
4.  **Pirámide Sólida:** Generar una estructura piramidal centrada.
5.  **Pirámide Invertida:** Dibujar una pirámide centrada con la base en la parte superior.
6.  **Diagonal Principal:** Imprimir una línea diagonal descendente (donde el índice de fila $i$ es igual al de columna $j$).
7.  **Diagonal Inversa:** Imprimir una línea diagonal ascendente.
8.  **Rombo (Diamante):** Generar un patrón de diamante simétrico.
9.  **V Invertida:** Dibujar solo el contorno de una pirámide (punta hacia arriba).
10. **Letra V:** Dibujar solo el contorno de una pirámide invertida (punta hacia abajo).
11. **Cuadrado Hueco:** Imprimir solo los bordes de un cuadrado, dejando el centro vacío.
12. **Secuencia Alternada:** Generar una línea de caracteres alternados por guiones bajos (ej: `X _ X _ X`).
